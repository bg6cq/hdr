# 手写数字识别神经网络

一个交互式 Web 应用，用于学习神经网络如何识别手写数字。

**纯前端** — 所有神经网络计算在浏览器中运行，无需后端。

## 网络结构

```
输入层 [784]  → 隐藏层1 [128]  → 隐藏层2 [64]  → 输出层 [10]
 28×28 像素      ReLU 激活       ReLU 激活       Softmax
```

## 使用方法

### 单步训练
1. 左侧画板手写一个数字
2. 勾选正确的数字标签
3. 点击「训练一次」
4. 观察中间激活图和右侧权重图的变化

### 批量训练（推荐）
1. 选择「每数字 X」— 每个数字取多少样本（20 起步，50 效果明显）
2. 点击「🚀 批量训练」
3. 观察准确率（acc）逐步上升

### 识别
- 切换到识别模式，画数字后自动预测
- 右侧概率条显示 0-9 每个数字的可信度

## 界面说明

| 区域 | 内容 |
|------|------|
| **左侧** | 手写画板 + 训练/识别控制面板 |
| **中间** | 网络激活可视化（输入→隐藏层1→隐藏层2→输出层的神经元激活值） |
| **右侧** | 第一层权重 W1（128 个 28×28 小图，展示每个神经元学到的特征） |

### 学习率
控制每步更新的幅度：**0.05–0.1** 推荐。太高会震荡，太低收敛慢。

### 权重图颜色
- **橙色** → 正权重（像素激活该神经元）
- **蓝色** → 负权重（像素抑制该神经元）
- **白色** → ≈ 零（无关）

## 训练数据

使用 MNIST 数据集（5000 张真实手写数字，每数字 500 张）。

## 键盘快捷键

| 键 | 功能 |
|----|------|
| Enter | 训练一次（训练模式下） |
| 0-9 | 快速选择数字标签 |
| C | 清空画板 |
| R | 识别（识别模式下） |
