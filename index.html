📖 使用说明

网络结构

输入层 784 (28×28 像素) → 隐藏层1 128 (ReLU) → 隐藏层2 64 (ReLU) → 输出层 10 (Softmax)。所有计算在浏览器本地完成，无需后端。

快速上手

  1. 点击「🚀 批量训练」，选择每数字 50 或 100 张真实 MNIST 样本训练网络。
  2. 等待训练完成（状态栏会显示实时准确率与损失）。
  3. 切换到「识别」模式，在左侧画板手写一个数字，自动预测。
  4. 右侧概率条显示 0–9 每个数字的置信度，最下方为大号预测结果。

单步训练

切换到「训练」模式：手写一个数字 → 勾选正确的数字标签 → 点击「训练一次」。可观察中间激活图和右侧权重图的变化。

权重 (Weights)

权重是连接两层神经元的强度，决定了输入如何影响输出。本网络共有 三组权重矩阵（外加偏置 b1/b2/b3）：

权重连接尺寸 (行×列)参数量
W1输入层 → 隐藏层1128 × 784100,352
W2隐藏层1 → 隐藏层264 × 1288,192
W3隐藏层2 → 输出层10 × 64640

尺寸约定：每行对应一个输出神经元，每列对应一个输入连接。例如 W1 是 128×784，即「128 个隐藏层1神经元，每个接收 784 个像素输入」。前向计算为 output = W · input + b。

权重图怎么看

右面板用顶部标签切换 W1 / W2 / W3，每个小图代表一个输出神经元的所有输入权重：

  • 橙色 → 正权重（该输入激活此神经元）
  • 蓝色 → 负权重（该输入抑制此神经元）
  • 白色 → ≈ 零（几乎无影响）

W1：每个神经元的 784 个权重天然排成 28×28，可直接看作「像素特征模板」——训练后常能看出笔画轮廓。W2 / W3：输入不是图像，故把 fan-in 个权重排成最接近的方形小图（W2 为 √128≈12×12，W3 为 8×8），仅表示权重强弱分布；W3 每个小图左上角标注了对应的数字 0–9。

每个小图独立归一化（除以自身最大绝对值），因此不同神经元之间颜色不可直接比较强弱，但能看清各自的权重结构。

手写输入预处理

手写笔画会经过 MNIST 标准预处理：裁剪笔画边界框 → 等比缩放到 20×20 → 按质心居中到 28×28。这使手写输入与训练数据分布对齐，是识别准确率的关键。

学习率

控制每步更新的幅度，0.05–0.1 推荐。太高会震荡，太低收敛慢。

权重图颜色

  • 橙色 → 正权重（像素激活该神经元）
  • 蓝色 → 负权重（像素抑制该神经元）
  • 白色 → ≈ 零（无关）
